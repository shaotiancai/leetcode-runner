# 动态规划
  
  本小节的题目主要关于动态规划算法，求解动态规划可以分解为以下五个步骤：
  1. 定义dp[i]/dp[i][j]的含义；
  2. 写出状态转移方程；
  3. 对dp数组进行初始化；
  4. 确定遍历顺序；
  5. 举例推导dp数组。

## 习题
  + NC68 跳台阶
    - dp[i], 到达第i级台阶总共有多少种跳法
    - dp[i] = dp[i-1] + dp[i-2]
    - dp[0] = 1, dp[1] = 1
    - 从左到右
  + NC19 连续子数组的最大和
    - dp[i], 包含下标i的最长连续子数组和
    - dp[i] = max(array[i], dp[i-1]+array[i])
    - dp[0] = array[0]
    - 从左到右
